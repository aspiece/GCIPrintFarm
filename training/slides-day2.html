How to Choose a Model That Prints

✅ Good signs

  • Mostly flat surfaces on the bottom
  • Thick walls and features (> 1.5 mm)
  • No tiny text or micro-details
  • Lots of positive reviews + photos
  • File type: 3MF or STL

⚠️ Watch out for

  • Lots of overhangs with no supports
  • Very thin walls or spires
  • Complex supports that may fail
  • File listed as "print at your own risk"
  • Extremely long print times (> 2 hours)
5 / 22
  • Use a real example: Pull up a model you know prints well, and one that commonly fails. Walk through both columns with those examples.
  • Overhangs explained simply: "Imagine printing a bridge — the middle of the bridge is in the air. If there's nothing below, the plastic just droops. Supports solve that but they add complexity."
  • Quick poll: "Raise your hand if the model you found has any overhangs or floating parts." Then help those students think through whether it needs supports.
  • Transition: "OK — before we can do anything with our model, we need to run it through the slicer. Let's learn what that actually means."
⏱ 4 min

What is a Slicer?

  • Software that converts a 3D model into printer instructions (G-code)
  • Breaks the model into horizontal layers — like slicing a loaf of bread
  • You control layer height, speed, infill, supports and more
  • Bambu Studio is our slicer — built specifically for Bambu printers
  • Without slicing, the printer has no idea what to build
🍞

Think of slicing a loaf of bread — you see each layer. The slicer does this to your 3D model and tells the printer how to build each one.

6 / 22
  • Key emphasis: Slicing is NOT optional. Every model must be sliced before printing. You can't just drag a file onto the printer.
  • Bread analogy tip: "If you could only eat a loaf of bread one slice at a time, you'd need to know in advance how thick each slice is, in what order they go, and where the butter goes. That's what the slicer figures out for the printer."
  • Common confusion: Students sometimes think Tinkercad does the slicing. It doesn't — Tinkercad only creates the 3D model file (STL/3MF). Bambu Studio does the slicing (creates G-code).
  • Transition: "Let's watch a video that shows Bambu Studio in action. This will make everything much more concrete."

📚 Vocabulary

Key Terms — Bambu Studio

Slicer Software that converts a 3D model into layer-by-layer G-code instructions for the printer. Bambu Studio is our slicer.
G-code Machine language the printer understands. Contains precise movement, temperature, and speed instructions for every moment of the print.
Infill The internal structure of a print. 15% = mostly hollow. 100% = solid. Higher infill = stronger but heavier and slower.
Layer Height How thick each printed layer is. 0.1 mm = high detail, slow. 0.3 mm = faster, less detail. 0.2 mm is the standard default.
7 / 22

  • Ask after reviewing: "If you were printing a keychain that needs to be strong, would you use 15% infill or 50%? Why?" (50% — more internal structure = stronger.)
  • Transition: "Keep these in your head while you watch. You'll need them for the guided notes activity afterward."

📚 Vocabulary

Key Terms — More Bambu Studio

Supports Temporary scaffolding for overhanging parts. Auto-generated by the slicer. Removed after printing — sometimes leaves marks.
Printer Profile The set of settings in Bambu Studio that matches your specific Bambu model (A1, P1S, X1, etc.). Must be correct for accurate results.
Preview Simulates the print layer-by-layer in Bambu Studio. Use it to verify support placement, check for gaps, and see estimated print time.
Slicing The process of converting a 3D model into G-code by dividing it horizontally into layers. Triggered by the "Slice" button in Bambu Studio.

🎓 Walkthrough

Step 1 — Open Your Model in Bambu Studio

Method A — Download and Open (Recommended)

  • Find your model on MakerWorld
  • Click Download → choose "Open with Bambu Studio"
  • Bambu Studio launches and loads the model automatically
  • ✅ No manual file import needed

Method B — Manual Import

  • Drag-and-drop an STL or 3MF file into the workspace
  • Or: File → Import → Local File
  • Model appears in the center of the workspace
  • Try the demo file: demo_plate.stl from the student page
13 / 23
  • Demo both methods on the projector. Start with Download and Open — navigate to MakerWorld, pick a model, click Download → Open with Bambu Studio. This is the preferred workflow.
  • Then show manual import using the demo_plate.stl from the student page — useful when MakerWorld isn't available or for the demo file practice.
  • Common issue: Student doesn't see "Open with Bambu Studio" option — they may need to set Bambu Studio as the default handler for .3mf files. Have a workaround ready (manual import).
  • Scaling check: Once imported, right-click → Check Size. For a keychain, the longest dimension should be ~40–80 mm.

🎯 Hands-On

Navigate the Workspace

🖱️ Mouse Controls

  • Rotate — Left-click + drag
  • Pan — Right-click + drag or middle-click + drag
  • Zoom — Scroll wheel (or Ctrl + scroll)
  • Reset View — Press R on the keyboard
🔄

Practice: rotate until you can see the bottom of the model, zoom in to see surface detail, then press R to reset.

💡 Getting comfortable with navigation now means fewer frustrations when you're slicing your real model.
14 / 23
  • Give students 2–3 minutes to try each control on their imported model (demo_plate.stl or their MakerWorld file).
  • Call out each move: "Everyone rotate — left-click and drag. Now zoom in with the scroll wheel. Now press R to reset." Do it together so nobody gets lost.
  • Common confusion: Students try right-click for rotate (that's pan). Clarify: left-click = rotate, right-click = pan.
  • Transition: "Good — now that you can see your model from any angle, let's adjust the print settings."

🎓 Walkthrough

Step 2 — Adjust Print Settings

Layer Height

  • 0.2 mm — default, good for most prints
  • 0.1 mm = higher detail, longer print
  • 0.3 mm = faster, less detailed

Infill

  • 15% = light, decorative
  • 30–40% = functional parts
  • Keychain: 20–25% is usually good

Supports

  • Turn ON if model has overhangs > 45°
  • Support type: Normal is fine for most
  • Remove supports carefully after printing
📌 For most class projects: 0.2 mm layer height · 20% infill · supports off is a solid starting point
14 / 22
  • Walk through settings on Bambu Studio live. Change a setting and show what it affects in the preview — this makes abstract concepts concrete immediately.
  • Key message: "You don't need to understand every setting right now. Start with these defaults. Once you've done a few prints, you'll know when and why to change them."
  • Common student mistake: Turning on supports when they're not needed (adds time and material). Ask "Does your model have parts that are floating in the air?" — if no, supports off.
  • Filament profile: Make sure students select the correct filament type in the slicer (PLA Basic, etc.) — wrong profile = wrong temperatures = bad print.
⏱ 5 min
🎓 Walkthrough

Step 3 — Slice & Preview

How to slice

  • Click the Slice button (top right)
  • Bambu Studio processes the model — takes 5–30 seconds
  • You're automatically taken to the Preview tab
  • Check the estimated print time (shown at bottom)

What to look for in Preview

  • Drag the layer slider to inspect layer-by-layer
  • Check supports are where you expect them
  • Look for gaps, unexpected holes, or missing walls
  • Verify the first layer is solid and complete
⚠️ Always preview before printing. 30 seconds of checking now saves 2 hours of failed printing later.
15 / 22
  • Live demo: Slice a model on the projector and walk through the Preview tab in real time.
  • Show a "bad" preview (if you have one): a model with a gap in layer 5 or missing supports — this makes the value of checking obvious.
  • Teach the layer slider: Drag from layer 1 upward — they should see the model build up visually. Ask "What does this remind you of?" (the FDM layer-by-layer concept from Day 1.)
  • Common student question: "The print time says 2 hours — is that right?" Sometimes yes. For a keychain at low detail it should be ~20–40 min. If it's way off, check scale and settings.

Spotting Problems Before You Print

🔴 In the model (Bambu Studio)

  • Gaps between layers in Preview → model may have holes/errors
  • No bottom layer → model may be floating above the plate
  • Extremely long print time → scale or complexity issue
  • Missing supports on obvious overhangs

🔴 During the print

  • First layer not sticking → bed alignment or temperature issue
  • No filament coming out → extruder jam or clog
  • Stringy webbing → retraction settings or wet filament
  • Print shifts mid-print → vibration or loose belt
18 / 22
  • This is a preview of troubleshooting skills they'll need on Day 5 (print day). Plant the seeds now.
  • Ask: "If you're in the middle of a print and you see stringy webbing all over the place — what's the first thing you'd check?" (Retraction settings or filament moisture.)
  • Key message: "The Preview tab catches half of these problems before you waste any filament. Use it every single time."
  • Watch for: Students who think they can just trust the slicer defaults without looking at the preview. Reinforce that preview is a required step, not optional.
